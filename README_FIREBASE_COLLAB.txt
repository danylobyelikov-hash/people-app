COLLABORATE (как Notes), код комнаты: TEAM

1) Открой https://console.firebase.google.com
2) Add project
3) Firestore Database -> Create database -> Start in test mode
4) Project settings -> Your apps -> Add app -> Web (</>)
5) Скопируй firebaseConfig
6) Открой firebase-config.example.js, вставь конфиг и сохрани как firebase-config.js
7) Залей на GitHub Pages файлы:
   - index.html
   - manifest.json
   - firebase-config.js
   - README_FIREBASE_COLLAB.txt (не обязательно)
   - firebase-config.example.js (не обязательно)

Готово: нажми "Общий доступ" -> отправь ссылку другим двум людям.
Они открывают ссылку, код уже TEAM (встроен), синхронизация включится автоматически.
